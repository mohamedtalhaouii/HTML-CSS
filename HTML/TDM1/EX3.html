Quelques définitions

    HTML

      Le HyperText Markup Language, généralement abrégé HTML, est un format de données utilisé pour représenter des pages web.

    CSS

      Les feuilles de style en cascade, souvent appelées CSS (de l'anglais Cascading Style Sheets), forment un langage informatique décrivant la présentation des documents HTML et XML.

    PHP

      PHP: Hypertext Preprocessor, souvent connu sous son acronyme récursif, est un langage de programmation open-source principalement utilisé pour générer des pages web dynamiques à travers un serveur HTTP.
